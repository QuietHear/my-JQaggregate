# 说明文档
> ### 使用步骤
>> 1. 下载插件
>> 2. 引入JQ
>> 3. 引入插件JS与CSS
>> 4. `<script>bigCarousel.start();</script>`载入即可
> ### 注意事项
>> * 指示器内容会自动生成，只需要添加指示器区域即可
>> * 轮播项至少2张才会有动画
>> * 该插件默认作用于`class=''bigCarousel`
>> * 图片地址等需要自己在*.HTML*中更换
> ### 兼容性
>> * IE8
>
